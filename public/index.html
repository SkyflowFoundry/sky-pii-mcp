For Existing Skyflow Customers

Connect to this MCP server using your Skyflow credentials. You'll need:

  • Bearer Token: Your Skyflow API bearer token
  • Vault ID: Found in your Skyflow dashboard
  • Vault URL: Your vault's base URL (e.g., https://your-id.vault.skyflowapis.com)
  • Account ID & Workspace ID: (Optional) Found in your Skyflow settings

Claude Desktop Configuration

Add this to your claude_desktop_config.json: 

{
  "mcpServers": {
    "skyflow-pii": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://pii-mcp.dev/mcp?vaultId={your_vault_id}&vaultUrl={your_vault_url}"
      ],
      "headers": {
        "Authorization": "Bearer {your_skyflow_bearer_token}"
      }
    }
  }
}

Authentication Model

This server uses bearer token pass-through authentication:

  • Your Skyflow bearer token is forwarded directly to the Skyflow API
  • No tokens are stored on this server
  • Each request can specify different vault configurations
  • Multi-tenant ready - multiple users can use their own credentials

API Endpoint

The MCP endpoint is available at:

POST /mcp

Query parameters:

  • vaultId - Your Skyflow vault ID (required)
  • vaultUrl - Your Skyflow vault URL (required)
  • accountId - Your Skyflow account ID (optional)
  • workspaceId - Your Skyflow workspace ID (optional)
